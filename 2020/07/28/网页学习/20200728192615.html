HTML+CSS3毛玻璃效果

小惠 · 2020-7-28 ·次阅读

开始

废话不多说，直接开始。

首先要新建一个HTML文档，然后给body添加一个背景图。

CSS代码：

body {
    background: url("background.png") center no-repeat fixed;
    background-size: cover;
}

然后我们的主角登场了，新建一个容器

HTML：

<div class="container"></div>

然后把东西放在容器里。

下面是重要的CSS代码：

.container {
    overflow: hidden !important;
    position: relative;
    z-index:10;/*一个比:before大的值*/
    background: inherit;
}
.container:before {
    content:"";
    width: calc(100% + 20px);
    height: calc(100% + 20px);
    background: inherit;
    box-shadow: inset 0 0 0 200px rgba(255,255,255,0.25);/*其中200px是根据.container的宽，最少等于二分之一的宽来定的，如果你给.container固定宽的话最好不过了*/
    position: absolute;
    top: -10px;/*这两个-10非常重要，是根据filter来定值的，模糊度为6，那么这个值最大为-6*/
    left: -10px;
    z-index: -1;/*比.container小*/
    filter: blur(6px) !important;
    overflow: hidden;
}

上面这些代码都是不可缺少的，其余样式由你自己来定，特别是宽高一定注意，否则可能会显示异常。

变换

同理，这样也可以套娃。

例如，不给body设置背景，而是给一个容器设置背景

<div class="box">
    <div class="small-box">
        1122333
    </div>
</div>
.box {
    width: 100vw;
    height: 100vh;/*撑满屏幕*/
    background: url("background.png") center no-repeat fixed;
    background-size: cover;
}
.smallbox {
    overflow: hidden !important;
    position: relative;
    z-index:10;
    background: inherit;
}
.smallbox:before {
    content:"";
    width: calc(100% + 20px);
    height: calc(100% + 20px);
    background: inherit;
    box-shadow: inset 0 0 0 200px rgba(255,255,255,0.25);
    position: absolute;
    top: -10px;
    left: -10px;
    z-index: -1;
    filter: blur(6px) !important;
    overflow: hidden;
}

这样就能简单实现效果了，如果上面代码有错误仍然不能实现效果，下面我给大家提供我自己写的代码（一个简单的登录页面），试过了能够正常显示（IE10以下的就别想了）
